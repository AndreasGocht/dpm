# Dresden Package Manager -- The package manager of last resort.

_This software is clearly in a pre-alpha stage. Use at your own caution._

Sometimes, especially in HPC, you are in a situation where you just need
to install some software in the most straightforward way.

This software hopes to be the most straightforward way.

## Usage

Packages can be installed using the `./dpm.py` utility.

Consult `./dpm.py --help` for help.

## Documentation

The `doc/` folder contains documentation for DPM in the Sphinx format.
